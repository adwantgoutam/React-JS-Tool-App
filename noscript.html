Troubleshooting

Your browser does not support advanced CMU features. Click here to retry loading all features.

Insight CMU GUI requires a Java™ runtime environment (JRE). If a JRE is not present on the machine running this browser, install one and restart the browser before trying to launch the Insight CMU GUI.

If you continue to face problems with launching GUI even after installing the JRE, try the following steps to manually launch Insight CMU GUI from the command line:

1) Download the Insight CMU GUI jar file to a folder (eg: /tmp or C:\cmugui) by right-clicking here and select "Save Link as" option from the context menu

2) From the command line, execute the downloaded jar file to launch the GUI.
Linux example:
java -jar -Xmx1024M /tmp/cmugui_standalone.jar
Windows example:
java -jar -Xmx1024M "C:\cmugui\cmugui_standalone.jar"

NOTE: If the java binary is not in PATH, then specify the full path of the java binary Eg: "C:\Program Files (x86)\Java\jre7\bin\java.exe" -jar -Xmx1024M "C:\cmugui\cmugui_standalone.jar"
